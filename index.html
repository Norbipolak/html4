Szelektorok(elemek kijelőlése)

ID (azonosító)

ID egy egyedi azonosító. Egy tagnek csak egy ID-ja lehet, egy id-t csak egyetlen tag kaphat meg.

Class (osztályba sorolás)

A class az osztályba sorlást jelenti. Egy tag-et lehet több osztályba sorolni is, egy tag kaphat több osztályt is.
